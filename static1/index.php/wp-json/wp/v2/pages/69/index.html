<\/p>\n\n\n\n

The goal of this projects was to create a datastructure in Java, which ist capable of calculating the gravity of over 10.000 astrological bodies with different physical properties like stars, planets and black holes, in real time. I used an octree structure to save all bodies and implemented the Barnes-Hut algorithm, which only looks at clusters of bodies and makes the program fast enough to run in real time.<\/p>\n<\/div>\n<\/div>\n\n\n\n
